1
Add the Apps Script to your Google Sheet
Open your Sheet → Extensions → Apps Script
Delete existing code, paste the entire Code.gs file → Save
2
Set your secret token inside Code.gs
In Code.gs find the line:
const SECRET_TOKEN = 'CHANGE_THIS_TO_YOUR_SECRET';
Change it to any password you choose, e.g. 'zaighum2025'
3
Deploy it as a Web App
Deploy → New deployment → Web app
Execute as: Me  ·  Who has access: Anyone
Click Deploy → Authorize → copy the URL
4
Paste your deployment URL and token below
Looks like: https://script.google.com/macros/s/…/exec
Security: Your token is stored only in this browser — never in the GitHub repo or code.
Anyone without the token gets rejected even if they find the Script URL.
Set Who has access: Anyone in the deployment (required to bypass Google's CORS block).
